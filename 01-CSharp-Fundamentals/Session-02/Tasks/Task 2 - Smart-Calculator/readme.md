# Task 2: Smart Calculator

## Objective
Create a calculator using conditions.

## Requirements
- Ask the user to enter two numbers
- Convert them into `double`

## Instructions
1. Ask for operation:
   - 1 → Addition
   - 2 → Multiplication

2. Use conditions:
   - If user chooses 1 → perform addition
   - If user chooses 2 → perform multiplication

3. Handle invalid input:
   - "Invalid operation selected."

## Output Example
Result = [value]
